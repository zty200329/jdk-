#源码阅读顺序
* 建议先从常用类进行入手进行解读,初次尝试源码阅读可以结合别人的博客进行阅读
* 具体可以先从方法和成员变量的作用进行理解和解读
* 然后进行方法内部的具体实现进行解读
* 最后结合子类和继承的父类或接口进行解读
